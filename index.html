How To Play

Place any unused bank letter on any empty square. If a row or column spells a valid allowed word, it scores positively. If a run is not valid, every tile in that run turns red and the full run scores negatively until you fix it. You can also select a placed tile and move it to any empty square. Trades remove one unused letter, add a new source word, and subtract that new word's value from your score.

How It Works

Three quick things before you start:

Tap a letter in the bank, then tap an empty grid square to place it. Tap a placed tile, then tap a new empty square to move it.
The source words that give you letters are banned as playable words, and you also cannot reuse close derivatives of them.
Valid words score positive points. Invalid runs turn red and score negatively until you fix them.
