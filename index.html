How To Play

Place any unused bank letter on any empty square. If a row or column spells a valid allowed word, it scores positively. If a run is not valid, every tile in that run turns red and the full run scores negatively until you fix it. You can also select a placed tile and move it to any empty square. Trades remove one unused letter, add a new source word, grow the board by one column, and subtract half that new word's value from your score.
